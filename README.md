# RH turnover

Ce projet est le quatrième dans la cadre de ma formation IA d'OpenClassroom. 
Il constitue un travail de ML sur une base de données de RH, où trois datasets ont été mergés, puis une analyse exploiratoire des données, et enfin, la recherche du modèle le plus performant pour prédire les départs.
Nous avons trouvé que le principal facteur de démission était les heures supplémentaires.
